# Execute in dev

1. clone repository
2. exec:

```
npm install
```
3. Nest cli installed
```
npm i -g @nestjs/cli
```
4. set the database
```
docker-compose up -d
```

5 clone file __tenv.template__ and rename it to __.env__ 

6 fill the variables for custom config

7 exec app:
```
npm run start:dev
```

8. rebuild database
```
http://localhost:3000/api/v2/seed
```
## Stack
- mongoDB 
- Nestjs


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
